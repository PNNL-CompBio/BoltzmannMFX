Documentation
=============

To build the BoltzmannMFX documentation locally, navigate into the ``doc`` directory and type:

.. code-block:: bash

   make

The documentation uses sphinx. You will need Python 3 with the following packages installed:

- sphinx_rtd_theme>=0.4.0
- recommonmark
- sphinx>=3.0
- pygments
- breathe
